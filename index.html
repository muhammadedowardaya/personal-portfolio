About Me

Halo, nama saya Muhammad Edo Wardaya, biasa dipanggil Edo atau bahkan MEDOW hihi_-

Saya bersekolah di SDN Tegallangkap 01

Di sini saya mulai menyukai untuk membuat sebuah gambar dan mewarnainya

Saya pernah berkesempatan untuk mewakili satu kecamatan dalam ajang lomba menggambar karena telah menjadi juara 1 pada lomba antar sekoklah SD satu kecamatan Tenjolaya, walaupun pada akhirnya saya pulang bukan sebagai seorang juara ahihihi

Di SMP PGRI Tenjolaya, saya mulai mengenal dunia komputer

Di SMK Putra Pelita Tenjolaya, saya memilih jurusan Teknik Komputer dan Jaringan, Di sini, saya semakin nyaman berinteraksi dengan komputer dan mulai mengenal Dunia Pemrograman Komputer

Setelah lulus dari SMK, saya membayangkan kembali akan seperti apa saya di masa depan, saya kembali melihat ke pemrograman komputer dan memutuskan untuk fokus ke 1 bidang yaitu Pemrograman Website dengan keterbatasan perangkat sebagai pertimbangan

Saya melanjutkan pendidikan ke STIKOM El Rahma dan mengambil jurusan Informatika, Di sini saya mendapatkan gambaran lebih luas tentang dunia hardware, jaringan komputer, robotik, bahasa pemrograman, struktur data, algoritma, database dan tentu saja dunia website.

Pada masa ini, saya banyak mengambil kursus online untuk memperkuat pondasi dalam membuat website. Saya belajar secara bertahap mulai dari html, css, tailwindcss, javascript, vite, reactjs, typescript, nextjs, dan mencoba berbagai library untuk membuat animasi seperti gsap dan melakukan kontrol pada 3D model menggunakan threejs atau 2D menggunakan pixijs

Saya adalah Front-End Web Developer yang berfokus pada pengembangan aplikasi web dengan fungsi dan alur penggunaan yang jelas.

Saya menikmati proses mengubah ide dan desain menjadi produk yang fungsional serta relevan bagi pengguna.

Saya memiliki pengalaman mengembangkan berbagai project mandiri menggunakan HTML, CSS, JavaScript, serta framework modern seperti React dan Next.js, dengan dukungan TypeScript, Astro, Tailwind CSS, dan Vite.
